# **Short-URL Lite**

## **介绍：**
> Short-URL Lite 是一个用 PHP 编写的短链接生成器。它不使用 MySQL，提供了一种简洁的短链接解决方案。支持语言包，比普通版轻便。
> 基于 Short-URL 3.1 版本
> 对比普通版的 848 KB，Lite版大小只有 8.75 KB 。降幅达到了 0.98968160 %（不算许可证与readme.md）

## tips:
> 普通版数据与Lite版数据不互通

## 功能：
* API 支持
* 语言包支持
